# Knee Connect

Knee Connect is a software system designed to support patients throughout the rehabilitation process following knee replacement surgery.
The project is designed with a modular structure, separating core functionality, graphical user interface, and documentation.

---

## Project Structure

- **Core Module**  
  Main processing logic and algorithms.

- **GUI Module**  
  User interface for interacting with the system.

- **Documentation & Materials**  
  Technical documentation, reports, and supporting materials.

---

## Technologies
- Python
- OpenCV
- MediaPipe
